Welcome to Your Electron App

This is a starter template for your Electron application. It includes:

Main Features

  • Custom Menu
  • Context Menu
  • electron-window-state for window state management

Custom Menu

We have implemented a custom menu for your Electron app. You can easily modify the menu items and structure according to your application's requirements. Check the code in your main process JavaScript file to customize the menu further.

Context Menu

Enhance user interactions with a context menu. Right-click on specific elements to trigger a context menu with relevant actions. You can customize the context menu items and functionality in your renderer process JavaScript file.

electron-window-state

We've included the electron-window-state module to manage the window state (position, size, and state) persistently. This helps in restoring the window to its previous state when the app is relaunched. Take advantage of this module to provide a seamless user experience.

Getting Started

To get started, update the content in the main section of this HTML file. You can add more components, styles, and scripts as needed for your application.

If you have any questions or need help, check out the documentation or community forums.
